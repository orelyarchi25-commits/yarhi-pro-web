# Yarhi Pro

A commercial SaaS application for aluminum contractors. Save time on calculations, generate quotes, and visualize projects with AR simulations.

## Getting Started

**Prerequisites:** Node.js 18+ and npm installed.

```bash
# Install dependencies
npm install

# Run development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Features

- **Landing Page** – Benefits, Login, and Start Free Trial
- **Dashboard** – Private area for calculator and simulation (requires mock login)
- **Mock Auth** – Click Login or Start Free Trial to simulate signing in; Logout to sign out

## Tech Stack

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Dark-blue & metallic color palette
