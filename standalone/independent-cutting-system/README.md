# Independent Cutting System (Standalone)

מערכת עצמאית לחלוטין שלא תלויה ב-Yarhi Pro.

## מה המערכת עושה

- הזנת מספר פתחים לפרויקט
- לכל פתח: סדרה (`7000`/`9000`), מספר כנפיים (`2`/`3`), רוחב, גובה
- חישוב רשימת חיתוך וחומר לפי תבנית
- חישוב זכוכית לכל פתח
- סיכום מצטבר לכל הפרויקט
- חישוב משקל לפי מק"ט (ק"ג/מ')

## הרצה

אפשר לפתוח ישירות את:

- `index.html`

או להגיש את התיקיה עם כל שרת סטטי פשוט.

## עריכת לוגיקה

כל הנוסחאות מוגדרות ב:

- `app.js` בתוך `TEMPLATE_LIBRARY`

מבנה תבנית:

- `rows`: רשימת פרופילים (מק"ט, שם, כמות, נוסחת אורך)
- `glass`: נוסחאות זכוכית

ערכי משקל:

- ברירת מחדל מוגדרת ב-`DEFAULT_WEIGHT_PER_M` בקובץ `app.js`
- אפשר לערוך בממשק בעמודת "משקל למטר לפי מק"ט"

משתנים זמינים בנוסחאות:

- `W` = רוחב פתח במ"מ
- `H` = גובה פתח במ"מ
- `N` = מספר כנפיים

> הערה: זו גרסת MVP. מומלץ לכייל נוסחאות מול דפי ייצור אמת לפני שימוש תפעולי מלא.
